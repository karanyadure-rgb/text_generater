# text_generater

A lightweight Python CLI tool that generates structured, customized content (articles, summaries, posts) tailored to an Indian audience. It leverages the lightning-fast **Groq API** and the `llama-3.1-8b-instant` model to generate responses and automatically saves them to a text file.

---

## 🚀 Features

* **Interactive CLI:** Prompts the user for a Topic, Content Type, Audience, and Tone.
* **Powered by Groq & LLaMA 3.1:** Uses Groq's high-speed API to deliver fast AI completions.
* **Auto-Save:** Automatically saves the generated content into a `.txt` file named after your topic.
* **Targeted System Prompt:** Pre-configured to write concise content (under 300 words) specifically for an Indian audience.

---

## 🛠️ Prerequisites

* Python 3.7+
* A [Groq API Key](https://console.groq.com/)
